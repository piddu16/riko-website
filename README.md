# Riko Website

The AI CFO for Indian SMEs. Chat with your Tally data in Hindi, English, or WhatsApp voice notes.

Live: https://rikoai.in

## Stack

- Next.js 16 (App Router) · TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React
- Deployed on Vercel

## Local development

```bash
npm install
npm run dev
```

Opens at http://localhost:3000

## Build

```bash
npm run build
npm run start
```

## Project structure

```
app/               # Routes (App Router)
components/        # Shared components
lib/site-config.ts # Brand + content
public/            # Static assets (logo, og-image, favicons)
```
